Our Mission

Why we built SheShield

SheShield was created as a final semester college project with one clear goal: to use technology for women's safety. We believe every woman deserves to feel safe — at home, in college, on the road, and online. This platform provides essential tools to help women stay informed and protected.

📚 About This Project

SheShield is a multi-page static website built using only HTML, CSS, and JavaScript — no external libraries or frameworks. It demonstrates core web development concepts including:

  • Responsive web design
  • Form validation using JavaScript
  • localStorage for data persistence
  • DOM manipulation
  • Event handling
  • Dark mode functionality
  • Arrays, Objects, and Loops in JavaScript

Technology Used

Built with basic web technologies

🌐

HTML5

Structure and content of all 6 web pages. Uses semantic HTML elements like nav, section, footer.

🎨

CSS3

Single shared stylesheet with CSS variables, flexbox, grid layout, and responsive media queries.

⚡

JavaScript

One shared JS file with functions, DOM manipulation, form validation, localStorage, and event handling.
